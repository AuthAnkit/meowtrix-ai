Project Description

MeowTrix-AI is a cutting-edge deepfake detection system that combines lightweight design with high accuracy. Using advanced computer vision techniques and machine learning algorithms, it provides reliable identification of synthetic and manipulated facial images.

Key Benefits

  • High accuracy detection with 94% success rate
  • Lightweight architecture requiring only 4GB RAM
  • Dual interface support for both CLI and GUI users
  • Real-time processing capabilities
  • Open-source and easily deployable

System Architecture

Processing Pipeline

1

Image Input & Preprocessing

Accept images in various formats, resize to 128x128 pixels, normalize brightness and contrast

2

Face Detection

Locate and extract facial regions using OpenCV Haar cascades or DNN-based detection

3

Feature Extraction

Extract LBP textures (10 histogram bins) and HOG gradients (1764 features) for comprehensive analysis

4

Classification

Combine features and classify using trained SVM/Random Forest models with confidence scoring

Feature Extraction Details

Local Binary Patterns (LBP)

Captures local texture information by comparing pixel intensities in neighborhoods, creating rotation-invariant patterns

Histogram of Oriented Gradients (HOG)

Extracts edge and gradient information, particularly effective for detecting structural anomalies in synthetic faces

Getting Started

Installation Instructions

1

Clone Repository

git clone https://github.com/yourusername/meowtrix-ai.git
2

Install Dependencies

pip install -r requirements.txt
3

Download Pre-trained Models

python download_models.py

Quick Start Commands

CLI Usage

python detect.py --input image.jpg --model svm

Basic deepfake detection on a single image

Batch Processing

python detect.py --input folder/ --batch --output results.csv

Process multiple images and save results to CSV

GUI Application

python gui_app.py

Launch the graphical user interface

Training Custom Model

python train.py --data dataset/ --model rf --features lbp+hog

Train a new model with your dataset
